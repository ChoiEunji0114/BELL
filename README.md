## 📕 Development Environment

- Xcode : 11.3.1
- Swift : 5.0.1
- cocoapods : 1.9.1
- Libraries
  - [Macaw](https://github.com/exyte/Macaw)

<br/>

## 📕 How to start

```
$ git clone https://github.com/ChoiEunji0114/BELL_iOS.git
$ cd BELL
$ pod install
```
